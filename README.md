# WebApi (ASP.NET Core MVC) – Chicago Bulls Demo

ASP.NET Core MVC уеб приложение (NET 8) с EF Core + SQLite и ASP.NET Core Identity (Register/Login). Проектът включва страници за **Privacy / Events / Products / Players**, тъмна/светла тема и примерна начална страница „Chicago Bulls“.

## Основни функционалности
- ✅ **Home** страница (Chicago Bulls) с hero секция + история/наследство
- ✅ **Products** – CRUD (Create/Read/Update/Delete) върху SQLite база с EF Core
- ✅ **Events** – страница/таблица (timeline)
- ✅ **Players** – страница/таблица
- ✅ **Аутентикация** – Register / Login с ASP.NET Core Identity
- ✅ **Dark / Light theme** (превключвател)

## Технологии
- ASP.NET Core MVC (.NET 8)
- Entity Framework Core 8
- SQLite
- ASP.NET Core Identity
